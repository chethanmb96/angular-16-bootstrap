# Angular 16 Bootstrap

A small Angular 16 application for managing products. It uses Bootstrap, reactive forms, and a json-server REST API backed by `db.json`.

## Run

```bash
npm install
npm run server
npm start
```

Run `npm run server` and `npm start` in separate terminals, then open `http://localhost:4200`.

The API runs at `http://localhost:3000/products`, and product data is saved in `db.json`.
